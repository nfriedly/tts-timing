Notes

Word Timings provided by the Watson Speech to Text service, essentially playing an automated game of telephone with the Text to Speech service. It's admittedly a bit crazy, but it does work.

Both options will play the audio and then log word timings to the console.

Server-side processing

  • Uses less client bandwidth, but uses significantly more server-side bandwidth
  • Allows for the audio to be captured and saved
  • Can provide faster-than-realtime results
  • Does not expose auth tokens (more secure)

Client-side processing

  • Offers easy, built-in html output
  • Uses more client bandwidth (to upload audio), but uses almost no server resources
  • May provide slightly degraded results due to additional lossy audio conversions on the client side
  • More limited browser support
